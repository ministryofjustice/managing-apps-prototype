Version 1.5

This version was created to use for research and usability testing focussing in on PIN phone applications.

Key pages and features:

  • Log an application
  • View an application details
  • View a list of pending and closed applications for a group
  • Record a decision for an application and log a reason
  • Forward an application to a different department
  • Leave a comment or update for an application that has not been processed

Detailed changes from previous version and page layouts in this figma.
